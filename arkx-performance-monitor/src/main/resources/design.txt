组件关系说明:
1. OSHI -> SystemInfoService: 提供硬件级系统指标
2. SystemInfoService -> SystemMonitor: 提供处理后的系统指标
3. TraceContext -> TraceRecorder: 提供跟踪上下文
4. TraceRecorder -> TraceRepository: 批量存储跟踪数据
5. MethodMonitorAspect -> TraceContext: 方法调用信息
6. Interceptors -> TraceContext: HTTP/SQL调用信息
7. SystemMonitor -> Config: 动态调整配置参数
8. Config -> 所有组件: 提供运行时配置
9. API Controllers -> 所有业务组件: 提供监控数据API
